AAML Duty Roster App
This is a web application for the American Advanced Medical Laboratory (AAML) to manage and view the monthly duty roster.
How to use:
 Open the application link provided by your supervisor.
 Use the search bar to enter your name.
 The application will automatically display your assigned role for today and your full schedule for the month.
For Admin Use:
 To update the monthly schedule, upload the new ⁠duty_roster.xlsx⁠ file to this repository.
 The application will automatically reflect the updates once the file is committed.
